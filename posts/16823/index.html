前端必备的实用工具
2019-12-02

工欲善其事，必先利其器。

前端工具

文档

  • devdocs：在线 API 文档，前端开发必备
  • miniprogram：小程序 API 文档，小程序开发必备

图标

  • iconfont：很全的 SVG 矢量图标
  • css.gg：纯 CSS 实现的图标

颜色

  • Flattastic Pro Color Palette：非常好看的色板
  • picular：配色界的 Google

布局

  • css grid generator：grid网格生成器

动画

  • animista：现成动画大全
  • perspective playground：3D 变换可视化
  • cubic-bezier.com：缓动函数可视化

图形

  • fancy-border-radius：自动生成border-radius图形
  • clippy：自动生成clip-path图形

阴影

  • shadows：box-shadow 柔顺阴影生成器

渐变

  • webgradients：渐变色集合
  • gradientfinder：能从图片中提取出渐变

SVG

  • URL-encoder for SVG：用于创建内联 SVG
  • SVG Filters：用于创建 SVG 滤镜效果
  • SVG Gradient Map Filter：用于给图片创建 SVG 滤镜效果
  • SVGOMG：优化 SVG 大小
  • SVG Path Editor：可视化编辑 SVG 路径

滤镜

  • Filter Generator：根据颜色生成其对应的滤镜效果

其他

  • three.js editor：three.js 的在线编辑器
  • loremipsum：文字占位符
  • picsum：图片占位符
  • carbon：将代码片段转化为精美图片用于分享
  • squoosh：优化图片大小
  • docsmall：在线压缩 gif
  • waifu2x：无损图片放大
  • remove.bg：去除图片背景
  • json2ts：将 json 转换为 ts 声明文件
  • qrbtf：艺术二维码生成
  • undraw：精美矢量插画
  • blobs：blob 图案生成
  • Which key：查询键盘按键对应的号码
  • font-face generator：将字体文件转化成编码
  • css-tricks：很多实用的 CSS 工具
  • caniuse：兼容性查询
